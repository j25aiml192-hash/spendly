# SpenDLy - AI-Powered SaaS Spend Intelligence

SpenDLy connects your finance, HR, and identity data to automatically detect waste, forecast renewals, and recover 20–30% of your SaaS budget within 90 days. It creates a single intelligence layer across all three, providing a clear and unified view for every team.

## Key Features

- **Automatic SaaS Discovery**: Connect your bank feed or accounting tool and Spendly maps every subscription within minutes—no manual entry.
- **Seat-Level License Intelligence**: See which employees haven't logged into a tool in 30, 60, or 90 days. Reclaim idle seats before renewal.
- **Duplicate Tool Detection**: AI clusters tools by feature overlap across departments (e.g., Figma vs Sketch, Slack vs Teams).
- **90-Day Renewal Forecasting**: Know which contracts are renewing, their risk level, and the recommended action weeks in advance.
- **Executive CFO View**: Get instant insights into projected savings, total spend, flagged waste, and actionable optimizations.
- **Interactive Dashboards**: Real-time filtering, active tool tracking, alerts, and detailed trend analytics.

## Tech Stack
- HTML5 / CSS3 / Vanilla JS
- Custom Design System (Premium Glassmorphic UI)
- Fully responsive across desktop, tablet, and mobile browsers

## Getting Started
To view the application locally:
1. Clone the repository.
2. Open `index.html` in your browser or run a local dev server (e.g., `npx serve .`).

## Application Structure
- `index.html`: Landing Page
- `dashboard.html`: Main Dashboard Overview
- `inventory.html`: SaaS Inventory with filtering
- `optimization.html`: AI-generated optimization recommendations
- `renewals.html`: Forecasting and renewal tracking Calendar
- `cfo.html`: Executive Summary and CFO metrics
- `settings.html`: Integrations and Team Management
- `license.html`: License Optimizer
- `spend.html`: Detailed Spend Analytics
- `duplicates.html`: Duplicate Detector
- `about.html`, `features.html`, `pricing.html`, `insights.html`: Marketing & Content pages
