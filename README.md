## 🌟 Overview

SpenDLy is a premium, enterprise-grade SaaS spend management application. Built for finance but designed for IT, it creates a single intelligence layer across your organization's tech stack. It intelligently maps subscriptions, identifies idle licenses, clusters duplicate tools, and forecasts upcoming renewals to ensure you never pay for unused software again.

### The Problem
Finance sees payments. IT sees users. HR sees employees. But no system connects them. Fragmented tools create comfortable guesswork rather than ruthless clarity.

### The Solution: SpenDLy
SpenDLy seamlessly integrates with your SSO logs, bank feeds, and HR systems to provide a unified, actionable dashboard that pays for itself.

---

## ✨ Core Platform Features

- 🔍 **Automatic SaaS Discovery**: Connect your bank feed or accounting software and SpenDLy maps every subscription across the enterprise within minutes—no manual data entry required.
- 👥 **Seat-Level License Intelligence**: Granular visibility into user engagement. See exactly which employees haven't logged into a tool in 30, 60, or 90 days, enabling targeted license reclamation.
- 🔀 **Duplicate Tool Detection**: AI-driven clustering analyzes feature overlap across departments (e.g., Figma vs. Sketch, Slack vs. Teams, Notion vs. Confluence) to eliminate redundant contracts.
- 📅 **90-Day Renewal Forecasting**: Proactive risk mapping. Know exactly which contracts are renewing, their associated risk levels, and receive AI-recommended actions weeks before the renewal date.
- 💼 **Executive CFO View**: A high-level, beautifully designed dashboard tailored for financial leadership, highlighting projected savings, total spend, and immediate optimization opportunities.

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge).
- A local web server to serve the HTML/CSS/JS files (e.g., Node.js `serve`, Python `http.server`, or Live Server extension in VS Code).

### Installation & Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/j25aiml192-hash/spendly.git
   cd spendly
   ```

2. **Serve the project locally:**
   Using Node.js:
   ```bash
   npx serve .
   ```
   *Alternatively, using Python:*
   ```bash
   python -m http.server 3000
   ```

3. **View the application:**
   Open your browser and navigate to `http://localhost:3000`

## 🛠️ Technology Stack

- **Markup**: Semantic HTML5
- **Styling**: Vanilla CSS3 with Custom Properties (Variables) for theming. Zero reliance on heavy CSS frameworks to ensure maximum performance and bespoke design control.
- **Interactivity**: Vanilla JavaScript (ES6+) for DOM manipulation, dynamic filtering, and UI state management.
- **Icons**: Lucide Icons (lightweight, clean SVG icon set).
- **Charts**: Chart.js for data visualization.
- **Fonts**: 
  - *Playfair Display* (Serif - Premium Headings)
  - *Outfit* (Sans-serif - UI text and readability)
  - *JetBrains Mono* (Monospace - Data and financial figures)
